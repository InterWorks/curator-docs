## /content/createFile

Creates a file model

Example Usage:

[Python Script](https://curator.interworks.com/file/api-example-createfile)

**Parameters:**
**file**

The file

**Returns:**

array

**Example Response:**

```JSON
    {
        "result": "Success",
        "msg": 0
    }
```

## /content/createNavMenu

Creates a nav menu model

**Returns:**

array

**Example Response:**

```JSON
    {
        "result": "Success",
        "msg": 0
    }
```

## /content/deleteFile

deletes a file model by ID

**Returns:**

array

**Example Response:**

```JSON
    {
        "result": "Success",
        "msg": 0
    }
```
